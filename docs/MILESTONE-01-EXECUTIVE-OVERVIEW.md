## Platform Vision & Mission

### Vision
**"To revolutionize building condition assessment and capital planning through intelligent, data-driven lifecycle management."**

Onyx Report aims to be the leading SaaS platform for facility managers, property owners, and capital planners to make informed decisions about building maintenance, renovation, and replacement strategies.

### Mission
Enable organizations to:
- **Assess** building conditions accurately using standardized methodologies
- **Predict** maintenance needs and failure risks proactively
- **Plan** capital expenditures strategically with data-driven insights
- **Report** facility conditions professionally to stakeholders
- **Optimize** building lifecycle costs and extend asset longevity

### Core Principles

1. **Standards-Based**: Built on Uniformat II classification system (industry-standard building element taxonomy)
2. **Data-Driven**: Quantitative FCI (Facility Condition Index) calculations for objective decision-making
3. **User-Centric**: Intuitive workflows designed for field assessors and facility managers
4. **Multi-Tenant**: Secure, scalable architecture supporting multiple organizations
5. **Actionable**: Transform assessment data into concrete maintenance and capital planning recommendations

## Problem Statement

### The Challenge

Organizations managing building portfolios face critical challenges:

#### 1. **Lack of Standardization**
- Inconsistent assessment methodologies across facilities
- Subjective condition ratings without quantifiable metrics
- Difficulty comparing buildings or tracking changes over time
- No industry-standard framework for element classification

#### 2. **Manual & Inefficient Processes**
- Paper-based assessments prone to errors and data loss
- Time-consuming manual data entry and report generation
- Limited photo documentation capabilities
- Difficult to track assessment history and trends

#### 3. **Poor Capital Planning**
- Reactive maintenance instead of proactive strategies
- Inability to prioritize repairs across portfolio
- Budget overruns due to unexpected failures
- No predictive analytics for maintenance timing

#### 4. **Communication Gaps**
- Assessors struggle to communicate findings to decision-makers
- Technical jargon confuses non-technical stakeholders
- Lack of professional, shareable reports
- Difficult to justify capital expenditure requests

#### 5. **Data Silos**
- Assessment data trapped in spreadsheets and documents
- No centralized repository for building information
- Historical data difficult to access and analyze
- Limited analytics and trend visualization

### Impact of These Problems

**For Facility Managers:**
- 40% more time spent on manual data compilation
- Increased risk of overlooked critical deficiencies
- Difficulty demonstrating ROI for maintenance budgets

**For Property Owners:**
- 25-30% higher lifecycle costs due to reactive maintenance
- Reduced asset values from deferred maintenance
- Compliance and safety risks from missed inspections

**For Capital Planners:**
- Inability to prioritize projects objectively
- Budget uncertainty and overruns
- Missed opportunities for cost-effective interventions

## The Solution

### What is Onyx Report?

**Onyx Report** is a comprehensive, cloud-based SaaS platform that modernizes building condition assessment and lifecycle planning through:

1. **Structured Assessment Workflow**
   - Pre-Assessment: Building information and element selection
   - Field Assessment: On-site condition rating with photo documentation
   - Automated FCI calculation and report generation

2. **Industry-Standard Framework**
   - Uniformat II building element classification (64 standard elements)
   - Standardized condition rating scales (Excellent to Poor)
   - FCI methodology for objective facility condition measurement

3. **Real-Time Data Management**
   - Cloud-based platform accessible from any device
   - Immediate data synchronization across team
   - Comprehensive building and assessment history

4. **Professional Reporting**
   - Automated PDF report generation with charts and photos
   - FCI calculations with maintenance cost projections
   - Stakeholder-ready presentations

5. **Analytics & Insights**
   - Building portfolio analytics and comparisons
   - Cost trend analysis and forecasting
   - FCI vs. building age correlation studies
   - Predictive maintenance recommendations

### Core Components

**Frontend (React SPA)**
- Single-page application with React Router
- Context-based state management (Auth, Organization)
- Responsive design for desktop, tablet, and mobile
- Progressive Web App capabilities

**Backend (Node.js API)**
- RESTful API with Express.js
- JWT-based authentication with refresh tokens
- Role-based access control (Admin, Manager, Assessor)
- Comprehensive error handling and logging

**Database (PostgreSQL)**
- Multi-tenant architecture with organization isolation
- 15+ tables supporting complete data model
- Foreign key constraints and referential integrity
- Indexed for optimal query performance

**External Services**
- Cloudinary: Image upload and CDN delivery
- Mailgun: Transactional email and notifications
- Sentry: Error monitoring and performance tracking
- Render.com: Cloud hosting and deployment

### Security Features

- **Authentication**: JWT access tokens (1 hour) + refresh tokens (7 days)
- **Authorization**: Role-based permissions at API level
- **Data Isolation**: Organization-level data partitioning
- **Encryption**: HTTPS/TLS for all communications
- **Input Validation**: Zod schemas + express-validator
- **Rate Limiting**: Protect against abuse and DDoS
- **Security Headers**: Helmet.js for HTTP security
- **SQL Injection Protection**: Parameterized queries with Knex
- **XSS Prevention**: Input sanitization and output encoding
